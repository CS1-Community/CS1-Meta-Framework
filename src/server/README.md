# cs1-game-server
___

## Description

NodeJS 3D Multiplayer Server Module for use with the **CS1 Game Engine**.

## Usage



In **Terminal** 
```
npm install --save cs1-game-server

```

or **package.json**
```
"dependencies": {
    "cs1-game-server": "^0.x"
  }
```

In **server.js**
```
require('cs1-game-server').start();
```
